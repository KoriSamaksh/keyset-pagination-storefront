👋 How to test the live-update behavior

This app browses 200,000 products, newest first, using cursor-based pagination — not page numbers. That means pagination stays correct even if new products are added while you're browsing.

To see this in action:

  1. Click Load More / Next a few times to browse normally.
  2. Click the Admin (gear) icon at the top-right of the page.
  3. Use Quick Add to instantly insert a few random products, or Manual Add to add one specific product by name/category/price.
  4. Keep clicking Load More / Next — you'll see no duplicates and no skipped products, even with new data arriving.
  5. Go back to Page 1 (or refresh) to see the newly added products appear at the very top, since they're newest.
